# @svag/lib

[![npm version](https://badge.fury.io/js/@svag/lib.svg)](https://npmjs.org/package/@svag/lib)

`@svag/lib` is A library for drawing SVGs with common methods.

```sh
yarn add -E @svag/lib
```
